SOARS Computational Thinking and Data Science

You have reached the home page of the 2018 SOARS Computational Thinking and Data Science workshops.

The workshops this year promise to be a ton of fun and are designed to provide you an introduction and exploration of the tools for using Python for computational thinking and scientific data exploration.

Throughout the summer you can come to this website to find out what’s new and interesting with the workshops (among other things). You will also want to make sure you spend some time on the Github page for the workshops.
